# 專業交易雷達量化決策評分指南

本指南詳細說明了目前交易雷達採用的 **100 分量化決策評分邏輯**、**建議入手標準**，以及此評分機制的**優缺點評估**。

---

## 一、 量化評分標準說明

系統最終呈現的「決策分數」（百分制）是由一個基準分 **50分** 開始，透過加減分機制（累計得分為 `sc`），計算後限制在 **5~99 分** 之間：
$$\text{最終得分} = \max(5, \min(99, 50 + \text{累計得分} \times 3))$$

以下為各維度的詳細加減分標準：

### 1. 大盤環境過濾器（三段式安全閥）
* **大盤強多**（加權指數處於月線與季線之上）：**+2 分**
* **大盤震盪**（加權指數跌破月線或季線其中之一）：**-3 分**
* **大盤空頭**（加權指數同時跌破月線與季線）：**-8 分**

### 2. 技術面與均線指標
* **站穩月線 + 趨勢明確**（Close > 20MA 且 ADX >= 25）：**+3 分**
* **站穩月線 + 趨勢盤整**（Close > 20MA 且 ADX < 25）：**+1 分**
* **跌破月線**（Close < 20MA）：**-2 分**
* **短期線轉強**（Close >= 5MA 且 5MA 上彎）：**+1 分**
* **短期線轉弱**（Close < 5MA 且 5MA 下彎）：**-1 分**
* **明日轉強門檻高**（Close < 明日 5MA 扣抵價）：**-1 分**
* **負乖離過大**（BIAS < -5%）：**+1 分**
* **觸及布林下軌支撐**（Close <= BB_DN * 1.02）：**+2 分**

### 3. 量能與成交量變動
* **溫和出量**（成交量比5日均量放大 1.2 至 2.5 倍）：**+2 分**
* **主力進場特徵**（成交量 > 5日均量 * 1.1 倍）：**+2 分**
* **箱體整理突破 + 爆量**（Box_Breakout 且 vol_ratio > 3）：**+2 分**
* **無量上漲**（未達均量）：**-1 分**
* **未帶突破的異常爆量**（未突破且 vol_ratio >= 4）：**-4 分**（慎防隔日爆量出貨）
* **盤中估計量未確認**（即時動能快篩中）：**-1 分**

### 4. 動能與近月波段表現
* **近月漲幅亮眼**（ROC_20 > 10%）：**+2 分**
* **近月跌幅過大**（ROC_20 < -5%）：**-2 分**（避免接墜落的刀子）
* **趨勢品質優異**（Momentum_Score >= 75）：**+2 分**
* **趨勢品質偏弱**（Momentum_Score <= 35）：**-2 分**

### 5. 基本面（營收與獲利）
* **月營收雙增**（MoM > 0% 且 YoY > 0%）：**+3 分**
* **月營收強勢年增**（YoY > 15%）：**+2 分**
* **EPS 為正**（有獲利）：**+2 分**
* **EPS 虧損**：
  * 技術面極強（強勢股）：**0 分**（視為轉機股）
  * 技術面一般：**-1 分**

### 6. 籌碼面與法人動向
* **法人大買**（單日合計 > 3000張 或 佔均量比 > 5%）：**+2 分**
* **法人微幅買超**（單日合計 > 500張）：**+1 分**
* **法人大賣**（單日合計 < -3000張 或 佔均量比 < -5%）：**-2 分**
* **法人微幅賣超**（單日合計 < -500張）：**-1 分**

### 7. 反轉型態與過熱警示
* **紅吞噬反轉**：**+4 分**（多頭）／ **+1 分**（盤整）
* **黑吞噬反轉**：**-3 分**
* **長下影線回測支撐成功**：**+2 分**
* **長上影線反彈遇壓**：**-2 分**
* **指標過熱（未處於強趨勢）**：
  * KDJ 高檔（J >= 80）：**-3 分**
  * RSI 過熱（RSI >= 75）：**-2 分**
  * 逼近布林上軌（Close >= BB_UP * 0.98）：**-2 分**
  * 正乖離過大（BIAS > 7%）：**-2 分**
* **其他風險調降**：
  * 多空訊號衝突高：**-3 分**
  * VIX 偏高（VIX >= 25）：**-2 分**
  * 資料信心度偏低：**-1 至 -2 分**
  * 整理後突破（10日內箱體）：**+2 分**

## 二、 建議入手評分標準

系統根據量化得分，將個股分為三個評級。以下是實戰操作上的配置與切入建議：

### 🟢 1. 強勢買進區（80 分以上）
> [!IMPORTANT]
> **入手評估：高勝率機會，適合積極布局**
* **特徵**：通常大盤處於安全環境，且個股基本面優良（雙增或高EPS）、主力法人籌碼積極進場，且技術面剛站穩月線或帶量突破。
* **操作策略**：
  * 配合系統給出的「明日交易計畫」，若隔日開盤「突破今日高點且量能延續」可順勢切入。
  * 必須嚴守系統算出的「失效價（停損價）」，通常設在跌破月線或今日低點。
  * 嚴格禁止在開盤「開高超過 3.5%」時盲目追高。

### 🟡 2. 偏多觀察區（45 ~ 79 分）
> [!NOTE]
> **入手評估：分批布局或等待關鍵突破**
* **特徵**：基本面佳但短期量能尚未放大，或是技術面極強但大盤處於「震盪環境（扣3分）」或籌碼有些微鬆動。
* **操作策略**：
  * **在大盤偏空/震盪時**，若個股拿到 70~79 分，通常已是場內最強防守標的，可分批（例如 1/3 倉位）試單。
  * 若是技術指標過熱（如 KDJ 過高）導致被扣分，建議等待回測 5MA 或 20MA 有防守時再行介入。

### ⚪ 3. 忽略區（45 分以下）
> [!WARNING]
> **入手評估：空手觀望**
* **特徵**：多空訊號高度衝突、趨勢偏弱（20MA之下）、基本面與籌碼面無亮點，或大盤處於大空頭環境下。
* **操作策略**：不予操作，維持空手，靜待評分重回 45 分以上。

## 三、 本評分邏輯的優缺點

### 👍 優點 (Pros)
1. **多空訊號衝突自動偵測**：若一檔股票雖然大漲，但上影線很長且 KDJ 嚴重過熱，系統會自動給予多空衝突扣分，避免落入追高陷阱。
2. **大盤動態降級機制**：三段式過濾器確保了在大盤覆巢之下（指數雙跌破月季線時），個股評分全面被壓低，系統會自動減少推薦名單，保護資金安全。
3. **策略型態豐富**：不會一味追求突破，對「回測支撐型」等安全係數較高的左側型態亦給予合適的加分。

### 👎 缺點與限制 (Cons)
1. **歷史回測的財務生存偏誤**：歷史回測勝率使用的是「今日的基本面快照」去給歷史訊號評分，導致過去的勝率數據可能略微偏高（Lookahead Bias）。
2. **極端行情鈍化**：在全市場全面噴出的極強多頭牛市，或是全面恐慌的無差別暴跌中，技術指標（如 RSI/KDJ 鈍化）可能會導致評分系統暫時失真。
3. **無券商即時下單深度整合**：本系統為「決策輔助工具」，無法得知當下真實市場掛單簿的流動性與滑價影響。
