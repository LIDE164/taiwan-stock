## 🐛 常見問題排查

### **問題：應用啟動報錯 `ModuleNotFoundError: No module named 'analysis_engine_v2'`**
```bash
# 解決方案：確認檔案在同目錄
ls -la analysis_engine_v2.py
# 如果缺失，從分支下載
git checkout improve/analysis-engine -- analysis_engine_v2.py
```

### **問題：掃描速度很慢**
- 正常現象，因為新引擎邏輯更複雜
- 首次掃描 100 檔預期 2-3 分鐘
- 後續會快速暫存在 session_state

### **問題：看不到回測結果**
- 確認股票資料有 14 天以上
- 查看控制台是否有報錯
- 某些新股可能無歷史資料

### **問題：信心度總是 0%**
- 可能是趨勢結構判定過於嚴格
- 檢查 ADX 和 200 日均線判定邏輯
- 可在 `analysis_engine_v2.py` 調整門檻
